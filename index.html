Cyber Security Time Table

Monday: Python Programming ▼

Focus: Learning the basics of Python, which is widely used in cybersecurity.

- Week 1-2: Variables, data types, loops, and functions.
- Week 3-4: File handling, error handling, and modules.
- Extra Practice: Write small scripts like password generators, file scanners, etc.

Tuesday: Networking Fundamentals ▼

Focus: Understand networking basics that are crucial for cybersecurity.

- Week 1-2: OSI Model, IP addresses, MAC addresses, and subnetting.
- Week 3-4: TCP/IP protocols (HTTP, DNS, DHCP), port numbers, and basic configuration.
- Hands-on Practice: Use tools like Wireshark to analyze traffic.

Wednesday: Web Development (HTML, CSS, and Basic JavaScript) ▼

Focus: Learn basic web development to understand how websites work.

- Week 1-2: HTML & CSS (creating basic web pages).
- Week 3-4: Basic JavaScript for front-end scripting.
- Extra: Analyze how websites function, looking at the structure and layout.

Thursday: Operating Systems (Linux Focus) ▼

Focus: Familiarize yourself with Linux, an essential OS in cybersecurity.

- Week 1-2: Linux file structure, basic commands, and navigation.
- Week 3-4: Permissions, shell scripting, and package management (apt, yum).
- Hands-on Practice: Use a Linux virtual machine (VM) like Kali Linux or Ubuntu.

Friday: Networking Tools and Security Basics ▼

Focus: Dive deeper into networking and basic security tools.

- Week 1-2: Networking tools like Nmap, ping, traceroute, and netstat.
- Week 3-4: Basic security concepts (encryption, hashing, firewalls).
- Hands-on Practice: Set up a virtual private network (VPN) or secure a network with a firewall.

Saturday: Web Security & Python Scripting ▼

Focus: Combine web development and Python to learn about security.

- Week 1-2: Basic web vulnerabilities (e.g., XSS, SQL injection).
- Week 3-4: Writing Python scripts to analyze web traffic or vulnerabilities.
- Extra: Explore common cybersecurity threats on websites and how to mitigate them.

Sunday: Review and Project Day ▼

Focus: Consolidate your week's learning by reviewing or working on small projects.

- Morning/Afternoon: Build a simple project (e.g., a basic website or a Python networking tool).
- Evening (9-11 PM): Review notes, practice commands/scripts, or learn new concepts like Virtual Machines (VMs) or containers (Docker).
